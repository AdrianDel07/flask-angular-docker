# flask-angular-docker


# Install 
before use the application, your could have some these applications:
- Docker
- Python3
- Flask
- Pep8
- Angular and Angular CLI


# Run Flask

```
docker build -t flask-api:latest .
```

```
docker run -d -p 5000:5000 flask-api
```
